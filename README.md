### 📅 FEB 26 2026.md — Advanced Matrix Types

| # | Problem | Approach | Key Logic |
|:--|:--------|:---------|:----------|
| 1 | Check Toeplitz matrix | `arr[i][j] == arr[i+1][j+1]` for all valid `i,j` | Diagonal equality propagation |
| 2 | Check diagonal matrix | Off-diagonal elements must all be 0 | `i != j && arr[i][j] != 0` → fail |
| 3 | Check identity matrix | Diagonal = 1, all others = 0 | Combined diagonal + off-diagonal condition |
| 4 | Create transpose | Swap row and column indices on output | Print `arr[j][i]` |
| 5 | Sum of lower triangular | `j <= i` traversal | Lower-triangle condition |
| 6 | Print boundary elements | Four separate passes (top, bottom, left, right) | Edge row/column traversal |
| 7 | Check sparse matrix | Count zeros and non-zeros, compare | `zero > nonzero` → sparse |
| 8 | Check scalar matrix | Diagonal constant = `arr[0][0]`, rest = 0 | Combined scalar check |
| 9 | Transpose then subtract | Build transpose matrix, then `arr[i][j] - arr1[i][j]` | Matrix difference |
| 10 | Transpose then add | Build transpose, then `arr[i][j] + tran[i][j]` | `printf("%02d")` formatting |

## ⚙️ How to Run the Code

All solutions are written in **Java**. To run any solution:

```bash
# Compile and run
javac Solution.java
java Solution

# For LeetCode-style class problems, wrap in a main method first
javac Main.java
java Main
```

Phase 1 problems are provided as **PDF files** — open them to view the problem statement and handwritten/typed solution side by side.

Phase 2 `.md` files contain numbered problems with complete Java code blocks — copy any block into your IDE or run directly on [OneCompiler](https://onecompiler.com/java).
